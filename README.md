# JRTech - Tecnología Humanizada

Este es el repositorio oficial de **JRTech**, liderado por Albeiro. Aquí se gestiona el catálogo de productos, la integración con IA para análisis de equipos y la generación automatizada de posters para redes sociales.

## 🌐 Ver Sitio en Vivo
Puedes acceder a la versión lista para usar aquí:
👉 **[https://albeirojr-pixel.github.io/jrtech/](https://albeirojr-pixel.github.io/jrtech/)**

---

## 🚀 Características
- **Catálogo Inteligente:** Carga automática desde Google Sheets.
- **Análisis con IA:** Veredictos y recomendaciones personalizadas para portátiles y celulares.
- **Posters Automáticos:** Generación de imágenes listas para Instagram/WhatsApp con un solo clic.
- **Seguridad:** Configuración privada para APIs y números de contacto.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Glassmorphism UI)
- JavaScript (Vanilla)
- Integración con OpenAI API & Google Apps Script
- Canvas para generación de posters
